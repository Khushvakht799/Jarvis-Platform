# Jarvis-Platform 🤖

Продвинутая AI-платформа для оркестрации MIG (эфемерных) процессов с визуализацией в реальном времени и интеграцией локальных языковых моделей.

## 🏗️ Архитектура

Jarvis-Platform/
├── backend/ # Rust ядро с gRPC API
├── services/ # Python оркестратор LLM
├── frontend/ # Electron интерфейс
├── storage/ # Артефакты и снапшоты
└── models/ # Локальные AI модели (GGUF)
text


## 🚀 Быстрый старт

### Предварительные требования
- Rust 1.91+
- Python 3.11+
- Node.js 18+
- Модели DeepSeek (GGUF формат)

### Установка
1. Клонируйте репозиторий:
\\\ash
git clone https://github.com/Khushvakht799/Jarvis-Platform.git
cd Jarvis-Platform
\\\

2. Установите зависимости:
\\\ash
# Бэкенд
cd backend && cargo build

# Сервисы  
cd ../services/local_llm_orchestrator
pip install -r requirements.txt

# Фронтенд
cd ../../frontend && npm install
\\\

3. Запустите платформу:
\\\ash
# Windows
./start-jarvis.bat

# Или вручную:
# Терминал 1: cd backend && cargo run
# Терминал 2: cd services/local_llm_orchestrator && python main.py  
# Терминал 3: cd frontend && npm start
\\\

## 🎯 Возможности

- **Оркестрация MIG процессов**: Эфемерные вычисления с автоматической очисткой
- **Визуализация в реальном времени**: Мониторинг процессов в Electron UI
- **Интеграция локальных LLM**: Модели DeepSeek для генерации кода и анализа
- **Векторные снапшоты**: Многомерное сохранение состояния
- **Многослойная архитектура**: Чистое разделение ответственности

## 🔧 Компоненты

### Бэкенд (Rust)
- gRPC API сервер
- Планировщик MIG процессов
- Управление памятью
- Движок векторных снапшотов

### Оркестратор (Python) 
- Управление локальными LLM
- Маршрутизация и распределение задач
- Загрузка моделей и инференс

### Фронтенд (Electron)
- Умный ввод с автодополнением
- Визуализация процессов в реальном времени
- Отображение и управление артефактами

## 🛠️ Разработка

### Добавление новых MIG процессов
1. Определите процесс в \ackend/src/runtime/\
2. Добавьте обработчик задач в \services/local_llm_orchestrator/\
3. Обновите визуализацию в \rontend/src/components/\

### Конфигурация
Редактируйте \config.json\ для:
- Путей к моделям и параметров
- Настроек runtime
- Расположения хранилищ
